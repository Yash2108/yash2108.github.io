Topic Classification

We first train a classifier on the existing labeled dataset. This gives us a base model. We use this base model to generate predictions of a portion of the unlabeled data. These predictions are also called pseudo-labels. We stack the pseudo labels and existing labeled data to generate a bigger dataset and train an improved classifier. This classifier is then used to further generate predictions and the process repeats till we get an acceptable result. We have implemented this approach in 2 ways:
